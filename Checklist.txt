O trabalho consiste em implementar uma loja virtual (só a aparência, a loja não vai "funcionar") utilizando as linguagens vistas na disciplina. O que será cobrado:

HTML: documentos válidos e acessíveis, imagens, tabelas
CSS: documentos válidos, layout responsivo (inclusive para mobile), deve usar o mínimo de classes e ids
Página inicial com menu de categorias de produtos e listagem inicial de produtos em destaque
Tela específica para detalhar apenas 1 produto clicado (faça pelo menos 2 produtos)
Tela do carrinho de compras (com pelo menos 2 produtos fictícios para teste)
Além de ser enviado aqui, o site deve ser hospedado e o link deve ser informado via comentário no envio ou arquivo txt
Outros sites que não sejam lojas virtuais, porém com complexidade similar, são aceitáveis, mas consulte o professor com antecedência.

Itens que serão avaliados:

HTML válido com imagens, listas, tabelas e multimídia X
